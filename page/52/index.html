vue项目npm run dev正常显示，npm run build打包之后没有内容

发表于 2019-02-22 更新于 2025-01-24 本文字数： 250 阅读时长 ≈ 1 分钟

我的vue项目npm run dev正常显示，npm run build打包之后没有内容。原来是我在配置路由的时候开启了mode: ‘history’

var router=new VueRouter({
    routes:routes,
    mode: 'history'//去除路径中的 # 号
})

关于去除URL中的#，请移步vue-router.

关键

因为这个导致打包后路径出问题。最后还是使用hash路径方式（将mode: 'history'注释了）完成这个项目。

常见参数
-a (all)显示所有选项，默认不显示LISTEN相关
-t (tcp)仅显示tcp相关选项
-u (udp)仅显示udp相关选项
-n 拒绝显示别名，能显示数字的全部转化成数字。
-l 仅列出有在 Listen (监听) 的服務状态

-p 显示建立相关链接的程序名
-r 显示路由信息，路由表
-e 显示扩展信息，例如uid等
-s 按各个协议进行统计
-c 每隔一个固定时间，执行该netstat命令。

提示：LISTEN和LISTENING的状态只有用-a或者-l才能看到
